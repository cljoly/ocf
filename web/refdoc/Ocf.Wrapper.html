val of_ok_error : (Yojson.Safe.json -> [ `Error of 'b | `Ok of 'a ]) -> Yojson.Safe.json -> 'a
of_ok_error f json applies f to json. If it returns `Ok x, then x is returned. Else Ocf.invalid_value is called with the given json. This function is useful to wrap *_of_yojson functions generated by ppx_deriving_yojson.
